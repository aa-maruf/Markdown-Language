- ## <ins>Heading</ins>
  ```markdown
    #  For H1 heading
    ##  Fr H2 heading
    ###  For H3 heading
    ####  For H4 heading
  ```  
- ## <ins>Quote</ins>
   > `>` sign is used for quote.
   >> `>>` is used for  nested quote.
- ## <ins>Comment</ins>
  ```markdown
      Syntax for comment:
     <!--- This is a comment --->
  ```

- ## <ins>Italic/bold / Italic & bold</ins>
   ```markdown
      <!--   Markdown syntax    -->
  
    _Italic_
  
    **Bold**
  
    __Bold__
  
    ***Italic & Bold***
   ```
   `Preview:`
   _Italic_
    **Bold**
    __Bold__
    ***Italic & Bold***
